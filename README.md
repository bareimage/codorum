# Codorum

A calm, focused micro-editor and file watcher for people who need clarity over clutter. Built for ADHD, sensory sensitivity, and anyone overwhelmed by noisy IDEs.

Drop files in. Watch changes unfold. Edit in peace.

## How it works

- Drag & drop files or folders to start watching
- Every save is captured as a snapshot with visual diffs
- Scrub through history to see how files evolved
- Organize with tabs, pin important files, search by name or content

## Keyboard shortcuts

| Shortcut | Action |
|---|---|
| ⌘S | Save active file |
| ⌘K / Right-click | Command palette |
| ⇧⌘↑ / ⇧⌘↓ | Switch between files |
| ⌘A | Select all files |
| Delete | Eject selected file |
| Escape | Clear selection |
| ← / → | Scrub history snapshots |
| L | Return to live view |

## Install

Download the DMG from [Releases](https://github.com/bareimage/codorum/releases).

4 themes available — n01z, paper, phosphor, ember.
